A sample magento module for practise

`GET /rest/V1/calculator/add/1/2`

`POST rest/V1/calculator/addMultiple`
```
{
    "numbers": [
        {
            "num1": 1,
            "num2": 2
        },
        {
            "num1": 4,
            "num2": 5
        }
    ]
}
```

`POST rest/V1/calculator/subtract`
```
{
    "number": {
        "num1": 100,
        "num2": 20
    }
}
```
